Search Instructions

Selecting Servers to Search

In the Search menu, click on "Select Storage Services". A menu box will appear, allowing selection of one or more MIRC storage services to search. If no sites are selected, the first site in the list is searched by default. To select more than one site, hold the "Ctrl" key down while clicking on additional sites.

Search Fields

Two search pages are available. The simple page contains only one free-text field. The full page contains multiple buttons allowing selection of groups of related fields as well as modifiers which define how many results are to be displayed in one search and how the results are to be displayed.

The fields on the search pages are used to search for documents containing information of specific types. MIRC selects documents that match all fields specified on the query page. If a field is blank, MIRC accepts all documents as matches for that field.

Each field may be composed of multiple search items together with boolean logic allowing complex searches to be defined. The Boolean operators are:

  • " " (space) is the logical and operator. A query field containing abc def matches only documents containing both abc and def (in any order) in that field.
  • "|" is the logical or operator. A query field containing abc | def matches documents containing either or both of abc and def in that field.
  • Items included in parentheses are logically grouped. A query field containing abc (def | ghi) matches documents containing abc plus either or both of def and ghi in that field.
  • Items included in quotations marks are grouped into one logical item. A query field containing "abc def" matches documents containing abc def as a single character sequence in that field.

The Free Text query field searches for items anywhere in a document, while the other query fields search only information identified within the document as being of the identified type.

Results per Site

On the full search page, the "results/site" field determines the maximum number of responses requested from each storage service which is searched. The "Next" and "Prev" links on the search results page allow the user to request additional groups of results from the storage services.
